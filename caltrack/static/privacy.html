Privacidad

Tus datos son tuyos. La web funciona de forma local y sin cuenta.

Volver a Caltrack

Datos en tu dispositivo

La web de Caltrack guarda en el navegador comidas, macros, fotos confirmadas, medidas, objetivos y entrenamientos. Las claves personales de xAI y Hevy también quedan en el almacenamiento privado de ese navegador y no se incluyen en copias o exportaciones.

No existe una cuenta de Caltrack, una base de datos propia en internet ni un servidor que reciba tu historial. Borrar los datos del sitio en Safari elimina esta copia local.

Fotos y xAI

Una foto de comida solo se envía directamente desde tu navegador a la API de xAI cuando eliges analizarla. Caltrack solicita que la petición no se conserve como historial. Debes revisar y confirmar la estimación antes de guardarla.

El entrenador trabaja localmente por defecto. Si conectas Grok y haces una pregunta, se envía un resumen limitado de objetivos, nutrición, medidas y entrenamientos recientes. No incluye fotos ni claves.

Las fotos de progreso nunca se envían a xAI, Open Food Facts o Hevy.

Apple Salud

Safari y las webs instalables no pueden leer Apple Salud. En esta versión el peso y otras medidas se añaden manualmente. El cliente nativo opcional, si lo instalas en el futuro, solicita permisos de HealthKit por separado y mantiene sus datos en el iPhone.

Hevy

Si añades una clave de Hevy Pro, el navegador consulta directamente la API de Hevy para obtener sesiones, ejercicios, series, cargas, repeticiones y RPE. Caltrack no actúa como intermediario y conserva localmente los entrenamientos que importes.

Strava no ofrece en esta web una conexión directa. Puedes registrar esas sesiones manualmente o usar el puente opcional de Apple Salud.

Copias, widgets y notificaciones

La copia JSON puede contener el historial y las fotos que hayas elegido guardar. El archivo queda bajo tu control. Nunca incluye las claves de xAI o Hevy.

Control y eliminación

Puedes borrar registros dentro de Caltrack, desconectar integraciones y revocar tus claves directamente en xAI o Hevy. Al no existir una cuenta remota, borrar los datos del sitio elimina el historial guardado en ese navegador.

Las estimaciones nutricionales no sustituyen consejo médico o nutricional.
